def agregar_utc_chile(df):
  df = df.assign(UTC = ['']*df.shape[0])

  cambios_hora = [datetime.datetime(2010, 1,1,0,0,0) , datetime.datetime(2013, 4, 28, 3,0,0), datetime.datetime(2013, 9, 8, 4,0,0), datetime.datetime(2014, 4, 27, 3, 0,0), datetime.datetime(2014, 9, 7, 4, 0, 0), 
                  datetime.datetime(2016, 5, 15, 3, 0, 0), datetime.datetime(2016, 8, 14, 4, 0, 0), datetime.datetime(2017, 5, 14, 3, 0, 0), datetime.datetime(2017, 8, 13, 4, 0, 0), datetime.datetime(2018, 9, 10, 1,1,1)]
  utc = 3

  for i in range(len(cambios_hora) - 1):
    inferior = cambios_hora[i]
    superior = cambios_hora[i+1]

    df.loc[(df['created_at'] <= superior) & (df['created_at'] > inferior), 'UTC'] = utc

    if utc == 3:
      utc = 4
    else:
      utc = 3
  
  return df

Conclusi√≥n

En base a lo experimentado, se puede ver como los clasificadores de los a√±os 2013, 2014 y 2015 tienen un desempe√±o peor que el clasificador original (del experimento 1). El clasificador del a√±o 2016 tiene un desempe√±o similar al original. Finalmente se puede ver como el clasificador del a√±o 2017 tiene un mejor desempe√±o que el clasificador original. Esta primera comparaci√≥n entre los clasificadores indica que en el a√±o 2017 las distintas zonas geogr√°ficas est√°n m√°s marcadas en cada twitt, permitiendo una mejor clasificaci√≥n de texto para las publicaciones de ese mismo a√±o.

Ahora se proceder√° a analizar el desempe√±o de cada clasificador entrenado con twitts de un solo a√±o frente a un dataset de prueba que contiene twitts de todos los a√±os. Se puede ver como todos los desempe√±os oscilan entre 0.47 y 0.49, distando bastante del desempe√±o original de los clasificadores de los a√±os 2016 y 2017. Con esto podemos decir que hay ciertas caracter√≠sticas que tomaron en cuenta estos √∫ltimos dos clasificadores al entrenarse que no se encuentran en el resto del dataset, caracter√≠sticas que se encuentran presentes solo (o con mayor frecuencia) en estos a√±os.

Finalmente podemos decir que los criterios usados para clasificar los twitts no son atemporales, ya que dependiendo del a√±o en el cual se entrene el clasificador se obtiene un desempe√±o distinto, sumado a que este desempe√±o no se replica cuando se clasifican twitts de otros a√±os.
